
# CBMo4ers Fullstack

This bundle includes:

- 🔁 Flask backend with mock data
- 🧠 Fully wired React frontend
- ⚙️ Tailwind and styled layout
- ✅ Works locally with Vite and Python

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
